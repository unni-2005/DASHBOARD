# Title
**Urban Pulse: City Livability Dashboard**
# Description

A dynamic Power BI dashboard analyzing city livability based on various metrics such as safety, mobility, air pollution, and accident analysis. The dashboard makes use of graphs, charts, etc., to visualize the performance of the city.

# Tools & Technologies Used
* Power BI
* Data Visualization
* Dashboard Creation
* Data Analysis
* Microsoft Excel/CSV Data Set
  
# Dataset used

* Bangalore Traffic Dataset
* Sample urban mobility data from CSV/Excel file

# Key Features

* Interactive Year Slider
* Air Pollution Analysis
* Area-based Accident Analysis
* Mobility Performance Analysis
* Safety Index Analysis
* Area-based Livability Analysis
